La Estrategia UX/UI Multicapa

Overview & stats

El producto opera en el mercado de la CDMX. Nuestro objetivo fue integrar tres perfiles de usuario completamente diferentes (PyMEs, SOFOMes y Banca Personal) en una experiencia unificada que combine la rigurosidad financiera con una navegación moderna y fluida.

Estrategia del Producto

Análisis del desafío

El desafío

Diseñar una sola plataforma que funcionara de forma relevante para tres perfiles muy distintos: PyMEs, SOFOMes y usuarios de banca personal.

Cada perfil tenía necesidades, flujos y niveles de complejidad financiera completamente diferentes entre sí.

La experiencia debía sentirse personalizada para cada tipo de usuario sin fragmentar el producto ni duplicar el trabajo de diseño.

Combinar la rigurosidad visual de una institución financiera con una interfaz accesible y fácil de usar para el usuario promedio.

Lo que se logró

Diseño de web y app móvil con experiencias adaptadas por perfil, mostrando módulos, dashboards y flujos relevantes según el tipo de cuenta.

Un solo sistema de diseño que soporta los 3 perfiles de usuario, manteniendo coherencia visual y reduciendo complejidad técnica.

Interfaz con apariencia de banca seria e institucional pero con UX simplificada, amigable e intuitiva para todo tipo de usuario.

Flujos diferenciados por perfil: crédito y nómina para PyMEs, gestión de cartera para SOFOMes, y pagos cotidianos para banca personal.

Perfiles Diseñados

User architecture

La arquitectura de información se divide en tres interfaces independientes, alimentadas por un único Design System unificado.

Perfil PyMEs

PyMEs

Módulos de crédito empresarial express, dispersión masiva de nómina para empleados, pagos automatizados a proveedores clave y monitoreo continuo del flujo de caja del negocio.

Perfil SOFOMes

SOFOMes

Gestión avanzada de carteras de crédito vigentes y vencidas, herramientas digitales de originación crediticia, seguimiento automatizado de cobranzas y reportes regulatorios ante la CNBV.

Banca Personal

Banca Personal

Pagos cotidianos de servicios públicos y privados, transferencias SPEI seguras instantáneas, alcancías digitales para metas de ahorro y gráficos de control de gastos cotidianos.

Especificaciones Técnicas

Diseñado sobre una estructura modular de componentes flexibles para permitir la personalización de la interfaz según el tipo de usuario sin duplicar recursos.

01 / Margen de Contenedor

Margen de 16px lateral simétrico para soportar una visualización segura en todo tipo de pantallas móviles de la CDMX.

02 / Coherencia de Marca

El color morado actúa como base estructural mientras que el dorado resalta los balances y las metas logradas.

03 / Toque y Accesibilidad

Botones de acción rápida con áreas generosas de interacción de 48px mínimos para evitar toques falsos en el transporte.
