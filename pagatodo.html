03

Estrategia del Producto

Análisis del desafío

El desafío

Diseñar una sola plataforma que funcionara de forma relevante para tres perfiles muy distintos: PyMEs, SOFOMes y usuarios de banca personal.

Cada perfil tenía necesidades, flujos y niveles de complejidad financiera completamente diferentes entre sí.

La experiencia debía sentirse personalizada para cada tipo de usuario sin fragmentar el producto ni duplicar el trabajo de diseño.

Combinar la rigurosidad visual de una institución financiera con una interfaz accesible y fácil de usar para el usuario promedio.

Lo que se logró

Diseño de web y app móvil con experiencias adaptadas por perfil, mostrando módulos, dashboards y flujos relevantes según el tipo de cuenta.

Un solo sistema de diseño que soporta los 3 perfiles de usuario, manteniendo coherencia visual y reduciendo complejidad técnica.

Interfaz con apariencia de banca seria e institucional pero con UX simplificada, amigable e intuitiva para todo tipo de usuario.

Flujos diferenciados por perfil: crédito y nómina para PyMEs, gestión de cartera para SOFOMes, y pagos cotidianos para banca personal.

Interfaz de la App Móvil

Mobile UI Showcase

Una experiencia nativa diseñada para brindar fluidez, control y transparencia. Cuenta con flujos adaptados dinámicamente según el perfil de cuenta ingresado (PyME, SOFOM o Banca Personal), incluyendo control de límites y dispersiones.

Dashboard Principal

Navegación Eficiente y Fluida

El home está diseñado para ofrecer una experiencia de usuario altamente intuitiva y sin fricciones. Su estructura organiza la información financiera y los servicios digitales en bloques visuales claros, permitiendo al usuario realizar consultas de saldo y operaciones frecuentes con total facilidad y rapidez desde una interfaz centralizada.

Seguridad & Wallet

Gestión de Seguridad Inteligente

Este módulo pone el control absoluto de la seguridad en manos del usuario mediante una interfaz intuitiva y eficiente. La sección permite gestionar la protección de la tarjeta de forma autónoma: desde el bloqueo instantáneo del plástico hasta la visualización protegida de datos sensibles como el CVV dinámico mediante autenticación biométrica, garantizando una experiencia de uso tan sencilla como segura.

Finanzas Personales

Gestión Financiera Inteligente

Este módulo transforma el análisis de datos financieros en una experiencia intuitiva y sencilla. Su interfaz simplifica la visualización del flujo de ingresos y egresos mediante gráficos claros y barras de progreso, permitiendo al usuario monitorear sus gastos por categoría y tomar el control de sus finanzas diarias de manera ágil y sin complicaciones.

Ingreso Programado

Relleno automatizado

Flujo inteligente que permite a los usuarios automatizar el fondeo de su cuenta secundaria mediante domiciliación programada. Los usuarios pueden rellenar su saldo cada mes desde su banco principal para realizar sus pagos de servicios cotidianos sin comprometer sus datos bancarios o tarjetas principales.
